Pokemon Game App

  • A dynamic Pokémon app fetches and displays Pokémon data from the PokéAPI, allowing users to explore and interact with their favorite creatures.
  • Users can drag and drop Pokémon images into a customizable container with a styled background.
  • Features include random positioning of Pokémon within the container and draggable interactions for a playful experience.

Tech: Java OOP Model & Controller | Pokemon API
